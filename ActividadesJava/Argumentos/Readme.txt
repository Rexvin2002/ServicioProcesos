MostrarArgumentos

Descripción:
------------
Este programa Java muestra los argumentos pasados por línea de comandos en la consola.
Cada argumento se muestra con su posición numérica correspondiente.

Autor:
-------
Kevin Gómez Valderas (2ºDAM)

Uso:
-----
1. Compilar el programa:
   javac MostrarArgumentos.java

2. Ejecutar el programa pasando argumentos:
   java MostrarArgumentos argumento1 argumento2 argumento3...

Ejemplo:
   java MostrarArgumentos hola mundo 123

Salida esperada:
   INICIO PROGRAMA
   0.-hola
   1.-mundo
   2.-123
   FIN DEL PROGRAMA

Características:
----------------
- Muestra una ventana de confirmación al inicio
- Lista todos los argumentos con su índice
- Formato claro de salida con separadores

Requisitos:
-----------
- Java JDK instalado
- Permisos de ejecución

Versión:
--------
1.0 (programa básico de demostración)
